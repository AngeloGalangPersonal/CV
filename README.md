# Angelo Galang Portfolio Website

This is a static one-page personal portfolio website.

## Files

- `index.html` - main website file
- `assets/` - profile photo and company logos
- `.nojekyll` - tells GitHub Pages to serve files as a normal static site

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this project, including `index.html`, `assets/`, and `.nojekyll`.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder, then save.
6. Wait a few minutes for GitHub Pages to publish the site.
